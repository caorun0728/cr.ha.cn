0.全局设置

git config --global user.name "曹润" 用户名
git config --global user.email "你的Email"

1、推送发布新的版本代码如下：

GitHub
git add .
git commit -a -m '0.1.297'
git push origin gh-pages
OSchina
git add .
git commit -a -m '0.0.03'
git push origin master
注：gh-pages为github，gitcafe-pages为gitcafe的。

2、克隆到本地
git clone git@github.com:caorun0728/caorun.biz.git
git clone git@github.com:<项目所有者名称>/<项目名>.git
git clone -b branch remote_repo

3、关于git软件设置

如何设置默认打开的目录
1）在软件安装目录内找到etc文件夹，然后打开profile文件。
2）添加代码如下：
# set Project Path
proj="你的项目文件目录"
cd $proj

5、其他相关语法

1、打开 cd
2、退出当前目录 cd ..
3、创建目录mkdir 文件夹名称
4、git remote # 仅显示远程仓库名称
5、git remote -v # 显示远程仓库名称和仓库地址
6、git remote show # 同git remote
7、git remote show origin # 查看远程仓库名称为 origin
8、初始化 git init
