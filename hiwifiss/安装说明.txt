极路由器安装shadowsocks插件

---------------

说明：本安装方法只能用在极路由器1S和2上，其他版本的极路由暂且不支持。

1、路由器必须开启开发者权限（官方备注，开启等于自动放弃保修政策）。
2、SSH登陆极路由器，可以使用putty或XSHELL软件登陆。
3、输入以下命令
cd /tmp && wget http://xiazai.u.qiniudn.com/shadowsocks.sh && sh shadowsocks.sh && rm shadowsocks.sh
4、登陆极路由后台192.168.199.1，高级设置中会有shadowsocks的选项，输入你的代理地址端口，保存开启。
需要注意的是：
A、请关闭极路由的自动更新功能，要不每次路由器升级后，会删除SS插件。
B、若shadowsocks选项显示的是：{ "msg": "请求的接口不存在.", "code": 560 }，请重启路由器。
5、由于内置脚本的智能代理过于老旧，所以需要更新。更新方法如下：
a、手动替换。
b、cd /etc/gw-redsocks/gw-shadowsocks && wget http://this.is26.com/download/gfw.txt && cat gfw.txt >> gw-shadowsocks.dnslist && /etc/init.d/dnsmasq restart
